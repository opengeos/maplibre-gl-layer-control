MapLibre Layer Control - Full Demo

Working

Features:

  • Layer Visibility - Toggle layers on/off with checkboxes
  • Opacity Control - Adjust layer transparency with sliders
  • Resizable Panel - Drag the width slider or use keyboard
  • Responsive - Panel positioned based on control location

Instructions:

  • Click the layers icon (top-right) to open the layer control
  • Demo layers: Fill, Line, Circle (points), and Raster overlay
  • Click the gear icon to edit layer styles
  • Use checkboxes to show/hide layers
  • Use sliders to adjust layer opacity
